# Learnify - Мобильное образовательное приложение

Learnify — это приложение, формирующее образовательные привычки через двухконтурную систему:
1. **Контур "Курсы"** — структурированное изучение тем по 5 направлениям
2. **Контур "Тренировка"** — адаптивное повторение карточек для закрепления знаний

## Технологии

- **Frontend:** Vue 3 (Composition API), Vite, TypeScript
- **UI Library:** Vuetify 3
- **State Management:** Pinia
- **Router:** Vue Router 4
- **Стилизация:** SCSS + Vuetify
- **Utils:** VueUse, date-fns
- **DevOps:** Docker, Docker Compose

## Установка и запуск

### Локальная разработка

1. Установите зависимости:
```bash
cd frontend
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Docker

1. Соберите и запустите контейнеры:
```bash
docker-compose up --build
```

2. Frontend будет доступен по адресу `http://localhost:3000`

## Основные экраны

- `/learning` - Обучение (список курсов и уроков)
- `/main` - Тренировка (интеллектуальное повторение карточек)
- `/achievements` - Достижения и статистика
- `/profile` - Профиль и настройки

## Функциональность

### Система карточек
- Теоретические карточки с объяснениями
- Практические карточки 4 типов:
  - Multiple Choice (выбор ответа)
  - Reflection (рефлексивные вопросы)
  - Case (мини-кейсы)
  - Text Input (текстовый ввод)

### Алгоритм Spaced Repetition
- Приоритет 1: Карточки с ошибками (needs_review)
- Приоритет 2: Карточки для повторения по графику
- Приоритет 3: Новые карточки
- Интервалы повторения: 1, 7, 16, 35 дней

### Геймификация
- Система опыта (XP) и уровней
- Стрики (серии дней обучения)
- Достижения за различные действия

- `npm run dev` - запуск dev сервера
- `npm run build` - сборка для production
- `npm run preview` - предпросмотр production сборки

Все типы данных определены в `src/types/index.ts`. Моковые данные находятся в `src/mocks/mockData.ts`.
